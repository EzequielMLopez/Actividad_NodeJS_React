Para que sirve el archivo package.json?
  El archivo package.json, sirve para configurar y describir cómo interactuar con la aplicación y ejecutarla.
  Permitiendo a npm iniciar el proyecto, ejecutar scripts, instalar dependencias, publicarlo en el registro de npm y realizar muchas otras tareas útiles.

Que diferencias notaron entre ejecutar un archivo directamente con NodeJS y levantar el servidor de React?
  Quizas debido a mi ignorancia respecto a estos temas o bien al no ser un proyecto extremadamente demandante, lo cierto, es que no note ninguna diferencia.
  A lo mejor, la mayor diferencia es en el procedimiento y mismo en que React ya te crear todo un entorno de desarrollo, listo para albergar una app
  con todas sus demandas y necesidades.
